# IRONBOX 기획서

> **IRONBOX** = 아이언맨(일론 머스크 별명) + 타임**BOX**
> 일론 머스크식 시간관리를 반영한 **개인 타임박스 다이어리 PWA**

---

## 1. 한 줄 정의
매일 아침 머릿속을 비우고(Brain Dump), 꼭 필요한 일만 골라(4D), 같은 일끼리 묶고(배칭), 오늘의 핵심 3개를 정해(Big 3), 30분 블록에 배치한 뒤(Time Box), 저녁에 돌아보는(Feedback · Plan vs Actual) **계획→실행→회고 루프 다이어리**.

## 2. 배경 / 사상
일론 머스크식 시간관리 + GTD의 실용 하이브리드.

| 요소 | 출처 | 설명 |
|------|------|------|
| **30분 타임박스** | 일론(원래 5분) | 하루를 30분 블록으로 쪼개 배치 (현실적 단위로 완화) |
| **Big 3** | 우선순위/크리티컬 패스 | 오늘 핵심 3가지 |
| **Feedback (Plan vs Actual)** | 일론의 피드백 루프 | 계획 vs 실제 시간 비교·회고 |
| **4D 제거** | first-principles | Do / Defer / Delegate / Delete — "이거 꼭 해야 하나?" |
| **배칭(Batching)** | 일론 | 같은 성격의 일을 묶어 연속 배치 |
| **Brain Dump** | GTD | 머릿속 할 일을 빠르게 비우기 |

## 3. 화면 구조 (단일 페이지, 위→아래 스크롤)
> 계획/회고 탭 분리 없음. 한 화면에서 하루 흐름대로.

1. **기상 직후 할 일** — 짧은 텍스트 한 줄 (매일 직접 입력)
2. **오늘의 Big 3** — 핵심 3가지(★로 선택), 완료 체크
3. **할 일 + 타임라인** (2단)
   - 할 일: Brain Dump 입력 → 4D 상태 / 배칭 태그 / ★Big3
   - 타임라인: 06:00–24:00 30분 블록, **드래그앤드롭 배치 / 모서리 드래그 길이조절 / 끌어서 이동**
4. **회고 (맨 아래)** — 계획 vs 실제(블록별 실제시간 수동 입력) + 통계 + 오늘 회고 메모
- 상단바: 날짜 이동(‹ ›/오늘), 달력, 검색, 백업 메뉴

## 4. 데이터 모델
```
Task {
  id, text,
  category,                       // 배칭 태그
  status,                         // do | defer | delegate | delete (4D)
  isBig3, done,
  plannedStart, plannedDur,       // 타임라인 배치(슬롯 인덱스, 30분 단위 길이)
  actualMin                       // 회고 때 수동 입력
}
Day { date(YYYY-MM-DD), wakeNote, tasks:[Task], feedback, updatedAt }
```

## 5. 저장 & 프라이버시 (핵심 정책)
- **데이터는 본인 구글 계정의 Firebase Firestore에 저장·동기화** (2026-06 전환). 보안규칙으로 **본인만 접근**, 타인·운영자 무단 열람 금지(일기 수집 금지 원칙 유지).
- 1차: **Firestore** `users/{uid}/days` — 구글 로그인 후 기기 간 실시간 동기화 + 오프라인 퍼시스턴스
- 추가 안전망(권장, 크롬/엣지 PC): **디스크 파일 자동저장**(File System Access API) + **JSON 내보내기/불러오기**
- 로컬 IndexedDB는 파일 핸들 등 메타 저장 용도만
- (배경) 초기엔 "로컬 전용·서버 전송 없음" 정책이었으나, 기기 간 동기화를 위해 **본인 계정 클라우드**로 전환 — 상세: [`docs/2026-06-01-firebase-backend-design.md`](docs/2026-06-01-firebase-backend-design.md)

## 11. 최신 변경 (UX 단순화)
- **4D 제거** → 할 일은 **완료 체크박스**만 (필터: 전체/미완료/완료)
- **Big 3 직접 입력** → Big3 칸에 입력하면 **할 일 목록에도 자동 추가**(★ 표시), ★로 제외 가능
- **미완료 이월** → 오늘 완료 못한 할 일은 다음 날 할 일로 **자동 이월**
- **인라인 편집** → 할 일·Big3 텍스트는 **더블클릭 수정**, "오늘의 나에게 하고 싶은 말"·"오늘 회고"는 더블클릭 편집 + **Enter 저장**(회고는 Shift+Enter 줄바꿈)
- **명칭 변경** → 기상 항목 = **"오늘의 나에게 하고 싶은 말"**
