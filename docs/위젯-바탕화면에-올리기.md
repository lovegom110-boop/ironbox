# IRONBOX 위젯 — 바탕화면에 올리는 법

위젯 화면(`widget.html`)은 이미 다 만들었고, 웹앱과 **같은 구글 계정·같은 데이터**를 씁니다.
이걸 "바탕화면에 항상 떠 있는 작은 창"으로 만드는 방법은 두 가지예요.

> **❓ 따로 받는 설치파일(.exe)은?** 없습니다. 이 위젯은 웹으로 도는 방식이라,
> "설치"는 **Edge가 대신**합니다 — 위젯 주소를 Edge로 열고 **"앱으로 설치"**를 누르면 그게 설치예요
> (시작메뉴 아이콘 생성 → 더블클릭으로 앱처럼 실행). 진짜 .exe 설치파일은 아래 **길 ②(Tauri)** 로
> 구워야만 나옵니다. 더 쉽게는, 같이 넣어둔 **`위젯 실행.bat`** 안의 주소만 내 위젯 주소로 바꿔
> 더블클릭하면 위젯이 앱 창으로 바로 떠요.

---

## 0. 먼저 지금 바로 확인 (로컬)
폴더에서 아래 실행 후 브라우저에서 위젯 주소 열기 → 구글 로그인하면 오늘 할 일이 떠요.
```
python -m http.server 8000
# http://localhost:8000/widget.html
```
체크·빠른추가가 웹앱/폰에 반영되는지 같이 확인.

## 길 ① 빌드 없이 (쉬움 · 추천 · 회사 PC에서도 OK)
프로그램을 따로 "굽지" 않고, 이미 있는 도구만으로 바탕화면 위젯을 만듭니다.

1. **배포**: 평소처럼 깃에 올리면 Vercel이 자동 배포 → `https://<내앱주소>/widget.html` 가 살아납니다.
2. **앱으로 설치**: Edge로 그 주소를 열고 → 주소창 오른쪽 **⊕(앱 설치)** 또는 `…` 메뉴 → **"이 사이트를 앱으로 설치"**.
   - 제목/아이콘이 "IRONBOX 위젯"으로 잡힌 작은 독립 창이 생깁니다.
3. **항상 위에 고정**: **PowerToys**(마이크로소프트 무료)의 **Always On Top** 기능 →
   위젯 창을 클릭하고 **`Win + Ctrl + T`** → 그 창이 항상 다른 창 위에 떠 있게 됩니다.
   - PowerToys 없으면: 윈도우 스토어/`winget install Microsoft.PowerToys` 로 설치.
4. 창 크기를 손바닥만 하게 줄여 화면 구석에 두면 끝.

> 장점: 설치할 거 거의 없음, **구글 로그인 100% 정상**(평소 Edge라서), 회사망에서도 됨.
> 단점: 위에 얇은 제목줄이 남고, '항상 위'는 PowerToys가 담당(앱 자체 기능은 아님).

## 길 ② Tauri 앱으로 굽기 (프레임 없는 예쁜 창 · 인터넷 되는 PC에서 한 번)
테두리 없는 진짜 위젯 창. 대신 **한 번 빌드**가 필요하고 그때 인터넷에서 도구를 받습니다.
회사망에서 막히면 **집 PC 등 인터넷 자유로운 곳**에서 구우세요. (구운 뒤 실행은 오프라인 OK)

1. **준비물 설치** (인터넷 필요, 1회):
   - Node.js(LTS), Rust(`https://rustup.rs`), 그리고 Windows면 WebView2(보통 윈도우11에 이미 있음).
2. **틀 만들기** — 이 프로젝트 폴더 안에 데스크톱용 하위폴더를 하나 만들어 공식 도구로 생성:
   ```
   npm create tauri-app@latest
   # 프레임워크: Vanilla, TypeScript: No 로 간단히
   ```
3. **창 설정만 아래로 교체** — 생성된 `src-tauri/tauri.conf.json` 의 창 부분을 이렇게:
   ```json
   "app": {
     "withGlobalTauri": true,
     "windows": [
       {
         "label": "widget",
         "title": "IRONBOX",
         "url": "https://<내앱주소>/widget.html",
         "width": 320, "height": 460,
         "minWidth": 280, "minHeight": 320,
         "resizable": true,
         "decorations": false,
         "transparent": true,
         "alwaysOnTop": true
       }
     ]
   }
   ```
   - `url` 에 **배포된 위젯 주소**를 넣는 게 핵심 — 진짜 https 주소라서 구글 로그인이 잘 됩니다.
   - 창 컨트롤(─/✕)·"자세히 보기"는 `widget.js`가 `window.__TAURI__`가 있으면 자동으로 네이티브 동작으로 붙습니다.
     (필요 권한: `core:window:allow-minimize/close/hide/start-dragging`, `shell:allow-open` — capabilities에 추가)
4. **굽기**:
   ```
   npm run tauri dev     # 미리보기
   npm run tauri build   # 설치파일(.msi/.exe) 생성
   ```
5. 윈도우 시작 시 자동 실행을 원하면 시작프로그램에 등록(또는 tauri autostart 플러그인).

> 장점: 테두리 없는 깔끔한 위젯, '항상 위'가 앱 자체 기능.
> 단점: 빌드 도구 설치·인터넷 필요. 버전이 달라 에러 나면 위 창 설정만 그대로 옮기면 됩니다.

## 추천
- **지금 당장, 회사 PC에서**: 길 ①.
- **나중에 더 예쁜 창을 원하면**: 인터넷 되는 PC에서 길 ②.
두 길 모두 같은 `widget.html` 을 쓰므로, 위젯 화면을 고치면 양쪽 다 같이 좋아집니다.
