# IRONBOX 바탕화면 위젯 — 작업 정리

> 작성: 2026-06-08 · 한 곳에서 보는 요약. 상세는 아래 링크 문서들 참고.

## 한눈에
- **무엇**: 오늘의 **Big 3 + 할 일**만 바탕화면에 떠 있는 작은 위젯. 보기·완료체크·빠른추가·전체앱열기.
- **방식**: 기존 IRONBOX 코드를 그대로 재활용한 **위젯 전용 화면**(웹) + 데스크톱 껍데기(Tauri).
- **데이터**: 웹앱과 **같은 구글 계정·같은 클라우드(Firestore)** → 웹/폰과 실시간 공유.
- **상태**: 위젯 화면 **완성·디자인 검증 끝**. 데스크톱 설치파일(.exe)은 **프로젝트+클라우드 빌드 준비 완료**, 빌드만 남음(배포 주소 필요).

## 2. 위젯이 하는 일 / 안 하는 일
- **한다**: 오늘 Big3·할 일 보기 / 완료 체크(웹·폰 반영) / `#태그` 빠른추가 / "자세히 보기"로 전체 앱 열기.
- **할 일이 많으면**: **할 일 목록만 스크롤**(날짜·Big3·추가칸·자세히보기는 항상 고정).
- **안 한다(=전체 앱에서)**: 30분 타임박스 배치, 달력, 회고, 노트장 등.

## 3. 데이터·로그인
- 위젯도 **같은 구글 계정으로 로그인** → 같은 Firestore(`users/{uid}/days`)를 봄.
- 위젯은 `onSnapshot`으로 **실시간 반영**: 웹/폰에서 바꾼 게 위젯에 즉시 뜸.
- 위젯에서 체크/추가 → 웹·폰에 반영(단, **웹앱 화면은 새로고침/날짜이동 시** 보임 — 현재 웹앱은 실시간 구독이 아니라서).
- 첫 로그인 1회만 하면 이후 세션 유지.

## 4. 디자인
- Samsung 라이트 톤 그대로: 화이트 + 삼성 블루(`#1428A0`) 포인트, Pretendard, 좌측 짧은 블루 바, 둥근 체크, 블랙 pill.
- "불러오는 중 → 로그인 → 위젯" 단계로 깜빡임 제거. 한글 줄바꿈 `keep-all`.
- 렌더링 캡처로 눈으로 확인 완료 → [../__widget_preview.png](../__widget_preview.png)

## 5. 바탕화면에 올리는 3가지 방법
1. **Edge "앱으로 설치"** (가장 쉬움, 설치파일 X) — 배포된 위젯 주소를 Edge로 열고 "앱으로 설치" → 시작메뉴 아이콘. 항상 위는 PowerToys `Win+Ctrl+T`.
2. **`위젯 실행.bat` 더블클릭** — 파일 안 주소만 내 위젯 주소로 바꾸면, 더블클릭 시 앱 창으로 바로 뜸.
3. **진짜 설치파일(.msi/.exe)** — `desktop/` Tauri 프로젝트로 빌드. **GitHub Actions 클라우드 빌드** 권장(내 PC에 Rust 설치 불필요).
